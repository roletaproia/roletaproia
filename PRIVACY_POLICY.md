# Política de Privacidade do Aplicativo RoboRoleta

**Última atualização: 11 de Novembro de 2025**

Esta Política de Privacidade descreve como o aplicativo móvel **RoboRoleta** (o "Aplicativo") coleta, usa e protege suas informações.

## 1. Coleta e Uso de Informações

O Aplicativo **RoboRoleta** é uma ferramenta de análise e estratégia para roleta online.

**O Aplicativo não coleta, armazena ou compartilha nenhuma informação pessoal identificável (PII) dos usuários.**

*   **Não Coletamos:** Nome, endereço de e-mail, número de telefone, localização geográfica, informações de pagamento ou qualquer dado que possa identificá-lo pessoalmente.
*   **Não Utilizamos:** Cookies, rastreadores de terceiros ou tecnologias de coleta de dados de usuário.

## 2. Dados de Uso (Não Pessoais)

O Aplicativo pode coletar dados de uso anônimos e agregados para fins de melhoria de desempenho e análise de falhas, como:

*   Modelo do dispositivo.
*   Versão do sistema operacional.
*   Interações anônimas com a interface do Aplicativo.

Estes dados são estritamente não pessoais e não podem ser usados para identificar o usuário individualmente.

## 3. Serviços de Terceiros

O Aplicativo utiliza a plataforma **Capacitor** para funcionar como um aplicativo híbrido. O Capacitor pode ter suas próprias políticas de privacidade e coleta de dados. Recomendamos que você revise as políticas de privacidade de qualquer plataforma de terceiros integrada.

## 4. Segurança

A segurança dos seus dados é importante para nós. Como não coletamos dados pessoais, o risco de violação de dados pessoais é minimizado.

## 5. Alterações a Esta Política de Privacidade

Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.
